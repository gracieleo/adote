## adote
### **Projeto criado na semana Pystack Week - O retorno**

- Aula 1 : 
Fluxo de dados do django |
Autenticação (cadastro, login)

- Aula 2 : 
Criação de tabelas no banco |
Upload de imagens | 
Listagem, filtro de dados

- Aula 3 : 
Url dinâmica |
Envio de email |
Dashboards
